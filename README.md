# AI Dashboard Generator

**Turn raw data into professional insights in seconds.**

This application leverages Generative AI to instantly design and build interactive data dashboards from simple CSV/Excel uploads. It automates the work of a data analyst, choosing the right charts, statistics, and layouts based on the data you provide.

## Value Proposition
*   **Zero Setup**: No complex configuration or SQL queries.
*   **Multi-Perspective**: Automatically generates Executive, Operational, and Analytical views.
*   **AI-Powered**: Intelligently names widgets and suggests relevant comparisons.
*   **Robust**: Handles messy real-world data gracefully.

## Getting Started
1.  **Backend**: Run the Python server.
    ```bash
    cd backend
    py -m main
    ```
2.  **Frontend**: Run the React interface.
    ```bash
    cd frontend
    npm run dev
    ```
3.  **Usage**: Open your browser to `http://localhost:5173`.
